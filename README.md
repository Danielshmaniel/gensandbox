This is the repository for my Passion Project Touhou Fangame:
# GenSandbox 幻サンドボックス
## A Terraria-inspired Reimagining of Gensokyo
An incident strikes Gensokyo where everything turns two-dimensional and creatures gain the power to manipulate blocks with ease. It's up to Reimu and her allies to save Gensokyo from its undoing!
